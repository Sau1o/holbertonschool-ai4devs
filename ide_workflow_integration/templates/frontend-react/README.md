# React Frontend Template with AI Support

## Features
- **AI Configuration:** Pre-configured Copilot rules for React Hooks and Functional Components.
- **Linting:** ESLint + Prettier setup for consistent styling.
- **Automation:** Includes script for auto-generating component documentation.

## AI & Automation Setup
- **Config File:** `.ai-config.json` (defines rules for the AI assistant).
- **Automation:** Uses `scripts/doc-gen.js` (simulated) to update prop-types documentation.

## Usage
1. Copy this folder.
2. Run `npm install`.
3. Use `npm run ai:doc` to trigger the documentation workflow.
