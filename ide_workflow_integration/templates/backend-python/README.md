# Python Backend Template with AI Support

## Features
- **AI Configuration:** Tuned for FastAPI/Flask with strict type checking.
- **Environment:** Docker-ready structure.
- **Automation:** Includes environment setup scripts and auto-test scaffolding.

## AI & Automation Setup
- **Config File:** `copilot-rules.yaml` (defines coding standards).
- **Automation:** `setup_env.sh` automates venv creation and dependency installation.

## Usage
1. Copy this folder.
2. Run `./setup_env.sh`.
3. AI is configured to suggest Pydantic models for data validation.
