# Spring Boot AI-Ready Template

## Overview
This template provides a standardized structure for Java backend services using Spring Boot, pre-configured for AI-assisted development.

## AI Configuration & Features
- **Frameworks:** Spring Boot 3.x, Lombok, JUnit 5, Mockito.
- **AI Context:** Pre-loaded `.copilot-instructions.md` to enforce Service/Repository patterns.
- **Code Style:** Google Java Style enforced via `.editorconfig`.

## Automation Setup
This template includes:
1.  **Test Scaffolding:** Scripts to auto-generate JUnit boilerplate.
2.  **Doc Generation:** AI triggers for Javadoc on public methods.

## Getting Started
1.  Clone this folder.
2.  Run `mvn clean install`.
3.  Open in VS Code to activate specific AI profiles.
