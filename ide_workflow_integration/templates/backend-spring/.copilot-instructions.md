# Copilot Instructions for Spring Boot

- **Architecture:** Always follow the Controller -> Service -> Repository pattern.
- **Lombok:** Use `@Data`, `@Builder`, and `@RequiredArgsConstructor` for dependency injection.
- **Testing:**
  - Use `Mockito.when()` for service mocking.
  - Use `MockMvc` for controller integration tests.
- **Security:** Never hardcode credentials; use `@Value("${property}")`.
