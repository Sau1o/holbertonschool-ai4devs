# UI Concept - Smart Home Hub

## Purpose
Enable users to control and monitor IoT devices from a centralized interface to optimize energy and security.

## Users
- Homeowners: Full administrative access
- Guests: Limited access to specific rooms

## Core Screens
- Dashboard with status overview
- Room Detail grid view
- Automation and Scenes setup

## Actions
- Toggle devices On/Off
- Adjust device attributes (Brightness/Temp)
- Create and edit Scenes
